# Machine Learning Term Project
## Suicide / Depression Text Classification

### Author:
Murshida Rahman Mouree

### Dataset:
Suicide and Depression Detection Dataset (Kaggle)

### Project Goal:
The goal of this project is to build a machine learning model to classify social media posts as suicide-related or non-suicide-related using text data.

## How to Run the Code

1. Open the notebook in Google Colab or Jupyter Notebook.
2. Run the following commands to download the dataset:

'!kaggle datasets download -d nikhileswarkomati/suicide-watch'
'!unzip -q suicide-watch.zip'

3. Run all cells sequentially from top to bottom.
4. The results (plots, model performance) will be displayed automatically.

### Requirements:
- Python 3
- pandas
- scikit-learn
- matplotlib
